# Typography

The library uses Tailwind's default type scale today (font sizes, weights, and
line-heights). A dedicated typography token scale is reserved as a placeholder in
`src/tokens/index.ts` (`typography`) until the design system formalizes its own.

The specimen below uses Tailwind text utilities over the theme's `foreground` /
`muted-foreground` tokens.
