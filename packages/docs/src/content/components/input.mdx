# Input

A single-line text field. A thin wrapper over `<input>` — it forwards its ref and
spreads every native attribute, so `type`, `value`, `onChange`, `disabled`, etc. all
work as expected.

## Usage

```tsx
import { Input } from '@midnite/ui';

<Input
  type="email"
  value={email}
  onChange={(e) => setEmail(e.target.value)}
  placeholder="you@example.com"
/>;
```

Always pair an input with a `<label>` (or `aria-label`) for accessibility.

Playground: `moon run ui:storybook` → **Components / Input**.
