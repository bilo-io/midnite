# Button

The primary action trigger. Five variants and four sizes, composed with `cva`.

## Props

| Prop | Type | Default |
| --- | --- | --- |
| `variant` | `default \| secondary \| outline \| ghost \| destructive` | `default` |
| `size` | `default \| sm \| lg \| icon` | `default` |
| …rest | `ButtonHTMLAttributes` | — |

## Usage

```tsx
import { Button } from '@midnite/ui';

<Button variant="secondary" size="sm" onClick={save}>
  Save
</Button>;
```

> **Do** use `destructive` only for irreversible actions. **Don't** stack two `default`
> buttons side by side — give the secondary action a `secondary`/`ghost` variant.

For the full interactive prop matrix and a11y checks, run the playground:
`moon run ui:storybook` → **Components / Button**.
