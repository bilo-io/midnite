# Tabs

A compact segmented control for switching between sibling views. **Controlled and
generic** over its value type, so the active value is a typed union, not a loose
string. The caller renders the matching panel.

<Demo>
  <TabsDemo />
</Demo>

## Props

| Prop | Type | Notes |
| --- | --- | --- |
| `options` | `TabOption<T>[]` | `{ value, label, icon? }` |
| `value` | `T` | active value (controlled) |
| `onChange` | `(value: T) => void` | required |
| `ariaLabel` | `string` | labels the `tablist` |

## Usage

```tsx
import { Tabs, type TabOption } from '@midnite/ui';

type View = 'board' | 'office';
const options: TabOption<View>[] = [
  { value: 'board', label: 'Board' },
  { value: 'office', label: 'Office' },
];

<Tabs<View> ariaLabel="View" options={options} value={view} onChange={setView} />;
```

The header's theme switcher on this very site is a `Tabs` instance.

Playground: `moon run ui:storybook` → **Components / Tabs**.
