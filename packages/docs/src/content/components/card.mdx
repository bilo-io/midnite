# Card

A surface container for grouped content, composed from `Card` + `CardHeader` +
`CardTitle` + `CardContent`. Uses the `card` / `card-foreground` tokens, so it sits a
step above the page background in both themes.

        Idle — waiting for the next ready task.

    </CardContent>
  </Card>
</Demo>

## Anatomy

```tsx
import { Card, CardHeader, CardTitle, CardContent } from '@midnite/ui';

<Card>
  <CardHeader>
    <CardTitle>Title</CardTitle>
  </CardHeader>
  <CardContent>…</CardContent>
</Card>;
```

| Part | Role |
| --- | --- |
| `Card` | bordered, rounded surface (`bg-card`) |
| `CardHeader` | top padded region for the title |
| `CardTitle` | uppercase, tracked label (`text-muted-foreground`) |
| `CardContent` | the body, padded |

Each part forwards its ref and spreads `div` attributes, so add `className` to tune
spacing or width.

Playground: `moon run ui:storybook` → **Components / Card**.
