# Media

## What it is

Media is your **generated-media library** — images and other assets, browsable in
one place and filterable by [project](/app/projects) and type. Every item is
self-documenting: it records the **prompt** and the **settings** it was generated
from, so you can see exactly how each asset was made and reproduce or tweak it
later.

## Purpose

Media keeps AI-generated visuals from scattering across individual
[sessions](/agents/sessions) and chats. Generate an image from a prompt, keep the
ones worth reusing, and pull them into projects or [slide decks](/app/slides) when
you need them. It's a durable, searchable home for output that would otherwise be
buried in a run's history.

## Creating one

Click **New media**, pick a type, and generate from a prompt.

| Action | How | Behind the scenes |
| --- | --- | --- |
| Create / generate media | **New media** → type picker → generate | `createMedia` |

## Listing & viewing

| Action | How | Behind the scenes |
| --- | --- | --- |
| List media | Media screen | `listMedia` |
| Filter by project & type | Header filters | `listMedia` |
| Open an item | Click it | `getMedia` → `/media/view` |

## Managing

| Action | How | Behind the scenes |
| --- | --- | --- |
| Update | Open → **Edit** | `updateMedia` |
| Delete | Item menu → **Delete** | `deleteMedia` |
