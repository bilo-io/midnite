# midnite design system & docs

This is **`@midnite/ui`** — the reusable component library and design system behind
midnite's web UI — documented in an app that is itself built entirely from it. The
shell you're looking at (header, sidebar, theme switcher, this content frame) renders
from the library's primitives and design tokens, so the site is its own proof that
the library is consumable outside the web app.

## What's here

- **Foundations** — the design tokens (colour, typography, radius) rendered live.
- **Components** — a page per primitive with usage guidance and live examples.

For the full interactive prop matrix and accessibility checks, each component page
links out to **Storybook** — the playground stays the source of interactive truth;
these docs are the narrative and canonical examples.
