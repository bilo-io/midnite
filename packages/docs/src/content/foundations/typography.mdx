# Typography

The system uses the platform sans-serif stack at Tailwind's default type scale today.
Typography is a **documented placeholder** in the token map (`tokens.typography`):
reserved so the system is structurally complete, to be formalised with bespoke
families / sizes / line-heights as the design system grows.

<TypeScale />

## Guidance

- **Body copy** is `text-base` / `leading-7` on `text-foreground`.
- **Secondary text** (captions, metadata) uses `text-muted-foreground`.
- **Headings** step down in weight and size; keep one `h1` per page.
- Prefer the semantic colour tokens for text, never a raw hue.
